# Overleaf project: The Measurement Problem as a Problem of Reference

This is an Overleaf-ready LaTeX project for the manuscript and its appendices.

## Compile target

- `main.tex` — manuscript with appendices and audit material
- `revision_memorandum.tex` — optional support memorandum listing exact claims, non-claims, and contents

Use **pdfLaTeX** on Overleaf.

## What the final package includes

- a manuscript title page and abstract
- the full main-body theorem ladder in public vocabulary
- a minimal branching diagram in the Everettian diagnosis section
- an interpretation matrix in the selector taxonomy section
- an objections-and-replies section in the main body
- a retrospective bridge appendix to the later admissibility corpus
- an audit appendix with dependency map and objection matrix
- an archive folder with source material and archival transcription
