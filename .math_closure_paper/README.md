# Foundational Rewrite Project - Step 6

This Overleaf-compatible project is the sixth drafting deliverable. It performs a formatting and visual-polish pass while keeping the manuscript in a journal-facing submission form.

## What Changed in This Step

- retitled the manuscript as **Fixed-Base Exhaustion for Internal Operators**;
- removed AMS running heads so the working PDF no longer prints the author surname at the top of each page;
- switched the document to a more journal-like Times-style text/math setup using `newtxtext` and `newtxmath`;
- preserved title and author names in headline/title case on the title page instead of AMS all-caps;
- lightly tuned section and theorem spacing for a calmer page rhythm;
- shortened and aligned the abstract with the new title;
- updated keywords and MSC year to the 2020 scheme;
- improved the flagship-families summary table by giving it a caption and cleaner spacing.

## Files to Compile

- `main.tex` - named author manuscript;
- `main_blind.tex` - anonymous submission variant.

Both files are intended to compile on Overleaf with the default pdfLaTeX workflow. The bibliography remains manual, so BibTeX is not required.
